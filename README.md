### 🎓 Student Pass/Fail Prediction
> **Python · scikit-learn · pandas · matplotlib · seaborn** · Data Analysis Course (273341)

Compared 3 ML classification models to predict student academic outcomes.

- Used **Student Performance Dataset** (UCI, 395 rows × 33 features)
- Removed G1/G2 to **prevent data leakage** (correlation > 0.8 with target)
- Applied **Label Encoding** and **Stratified 80:20 Train/Test Split**
- Identified `failures` (17.9%) and `absences` (11.3%) as top predictors via Random Forest
- Analyzed **class imbalance** impact; proposed SMOTE as improvement strategy

| Model | Accuracy | F1-Fail |
|---|---|---|
| Logistic Regression | **67.09%** | 38.1% |
| Decision Tree | **67.09%** | 35.0% |
| Random Forest | 64.56% | 17.6% |
